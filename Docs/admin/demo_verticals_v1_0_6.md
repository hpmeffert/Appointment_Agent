# Demo Verticals v1.0.6 Admin Guide

Version: v1.0.6
Audience: Admin
Language: EN

## What this release adds

This release adds one-click vertical demo preparation for:

- `Dentist`
- `Stadtwerke Wallbox`
- `District Heating`

## What happens on click

The operator chooses one vertical button.
The cockpit then prepares:

- vertical-specific sample titles
- vertical-specific presenter framing
- matching Google demo generation input through the existing demo calendar API

## Main route

- `http://localhost:8080/ui/demo-monitoring/v1.0.6`
