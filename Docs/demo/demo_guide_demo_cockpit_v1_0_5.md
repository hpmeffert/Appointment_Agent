# Demo Cockpit v1.0.5 Demo Guide

Version: v1.0.5
Audience: Demo Presenter
Language: EN

## Suggested flow

1. Start on `Dashboard`.
2. Show the top navigation and explain that this now feels like a real operator cockpit.
3. Switch language once to show localization.
4. Switch theme once to show day/night mode.
5. Open `Monitoring` to show the technical page.
6. Open `Settings` to show that demo controls and defaults are visible in one place.
