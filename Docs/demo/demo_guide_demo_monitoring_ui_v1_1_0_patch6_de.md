# Demo Guide Demo Monitoring UI v1.1.0 Patch 6

Version: v1.1.0-patch6
Audience: Demo
Language: DE

## So erklaerst du Patch 6 am besten

Dieser Release macht die Google Demo-Erzeugung genauer.

Du waehlst jetzt:

- ein Startdatum
- ein Enddatum
- einen Termin-Typ

## Einfacher Demo-Ablauf

1. Oeffne `Google Demo Control`
2. Setze `From Date`
3. Setze `To Date`
4. Waehle einen `Appointment Type`
5. Nutze `Prepare Demo Calendar` oder `Generate Demo Appointments`

## Was du sagen kannst

- Der Datumsbereich bestimmt, wann Demo-Termine auftauchen duerfen.
- Der Termin-Typ bestimmt, welche Business-Inhalte die Termine verwenden.
