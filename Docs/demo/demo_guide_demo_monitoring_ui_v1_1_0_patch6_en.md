# Demo Guide Demo Monitoring UI v1.1.0 Patch 6

Version: v1.1.0-patch6
Audience: Demo
Language: EN

## Best way to explain patch 6

This release makes Google demo generation more precise.

You now choose:

- a start date
- an end date
- an appointment type

## Simple demo flow

1. Open `Google Demo Control`
2. Set `From Date`
3. Set `To Date`
4. Pick an `Appointment Type`
5. Use `Prepare Demo Calendar` or `Generate Demo Appointments`

## What to say

- The date range decides when demo appointments may appear.
- The appointment type decides what kind of business content the appointments use.
