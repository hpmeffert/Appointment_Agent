# Release Notes v1.0.4 Patch 2 (DE)

Version: v1.0.4 Patch 2
Status: veroeffentlichte Patch-Linie
Language: Deutsch

- Erweitertes technisches Monitoring in der Demo-UI hinzugefuegt.
- Eine detaillierte Event-Timeline mit Zeitstempel, Event-Typ, Journey-ID, Correlation-ID und Trace-ID hinzugefuegt.
- Correlation- und Trace-Gruppierung fuer tiefere Request-Verfolgung hinzugefuegt.
- Last- und Concurrency-Simulation fuer `20`, `100` und `1000` Nutzer hinzugefuegt.
- Ein Performance-Panel mit durchschnittlicher Antwortzeit, maximaler Antwortzeit und Events pro Sekunde hinzugefuegt.
- Ein technischer Modus fuer rohe JSON-Payloads und Command-Strukturen hinzugefuegt.
- Tests laufen gruen.
- Docker-Laufzeit und Smoke-Test wurden erfolgreich verifiziert.
