# Release Notes v1.0.6 Patch 1

- Eine eigene `v1.0.6-patch1` Cockpit-Route fuer die verbesserte Vertical-Demo-Linie wurde hinzugefuegt.
- Die Termintitel fuer Zahnarzt, Wallbox und Nahwaerme wurden so verbessert, dass sie wie echte Kundentermine klingen.
- Vertical-spezifische Beschreibungen, Erinnerungstexte, Folgeaktionen und Kundenkontexte wurden ergaenzt.
- Die Cockpit-Payload wurde erweitert, damit Operatoren Zielgruppe, Beispieltermine und Szenario-Sprache besser erklaeren koennen.
- Automatisierte Tests fuer Vertical-Inhaltsqualitaet und Routensichtbarkeit wurden hinzugefuegt.
