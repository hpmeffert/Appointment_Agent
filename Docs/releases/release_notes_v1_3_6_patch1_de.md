# Release Notes v1.3.6-patch1

Version: v1.3.6-patch1
Language: DE

## Zusammenfassung

`v1.3.6-patch1` ist ein kombiniertes Dokumentations-Release.

Es verbindet:
- das Incident-artige Demo-Cockpit aus `v1.2.1-patch4`
- die Reminder- und Google-Demo-Story aus `v1.3.6`

## Was sich geaendert hat

- neuer kombinierter Menuepunkt fuer die Reminder- und Google-Demo
- Patch-4-Cockpit-Stil bleibt stabil
- einfache Doku fuer den kombinierten Operator-Ablauf
- wichtige Parameter werden in klaren Worten erklaert
- drei kurze Demo-Storys sind enthalten
- Version bleibt im Header und in der Help sichtbar

## Wichtige Parameter

- `silence_threshold_ms`
  Standard ist `1300 ms`.
- `reminder_offsets_minutes`
  Die Reminder-Zeiten vor dem Termin.
- `google_calendar_id`
  Der Google-Kalender fuer den verknuepften Termin.
- `appointment_type`
  Der Termin-Typ in der Demo.
- `job_status`
  Der Zustand eines Reminder Jobs.

## Was in diesem Release enthalten ist

- das Incident-artige Cockpit hat jetzt einen zusaetzlichen Menuepunkt fuer die `v1.3.6` Reminder- und Google-Demo
- der User Guide erklaert das kombinierte Cockpit in einfachen Worten
- der Demo Guide enthaelt drei kurze Storys
- der Admin Guide erklaert den neuen Menuepunkt und die wichtigsten Parameter
- die Release Notes zeigen die neue kombinierte Release-Linie

## Verifikation

- Header zeigt `v1.3.6-patch1`
- Help zeigt `v1.3.6-patch1`
- das Cockpit hat den neuen Reminder-/Google-Menuepunkt
- die Doku erklaert die Parameter in einfacher Sprache
