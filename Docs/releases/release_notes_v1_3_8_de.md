# Release Notes v1.3.8

- Eine bus-sichere Reply-to-Action Engine fuer eingehende LEKAB-Antworten wurde hinzugefuegt.
- Aktionskandidaten wie Absage, Bestaetigung, Verschiebung, naechste Woche, naechster Monat, naechster freier Slot und Slot-Auswahl wurden ergaenzt.
- Interpretationsstatus und Sicherheitswert wurden im Message Monitor sichtbar gemacht.
- Eine versionierte kombinierte Demo-UI-Route `/ui/demo-monitoring/v1.3.8` wurde hinzugefuegt.
- Eine versionierte eigenstaendige Reminder-Cockpit-Route `/ui/reminder-cockpit/v1.3.8` wurde hinzugefuegt.
