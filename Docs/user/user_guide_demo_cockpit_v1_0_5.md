# Demo Cockpit v1.0.5 User Guide

Version: v1.0.5
Audience: User
Language: EN

## What you can do

This cockpit has four main areas:

- `Dashboard`
- `Monitoring`
- `Settings`
- `Help`

## What the switches mean

- `EN / DE`
  Switches the visible language.

- `Day / Night`
  Switches the cockpit theme.

- `Dashboard mode`
  Lets you focus more on the story, more on monitoring, or keep both balanced.
