# User Guide Demo Monitoring UI v1.1.0 Patch 6

Version: v1.1.0-patch6
Audience: User
Language: DE

## Google Demo Control hat jetzt drei Hauptfelder

- `From Date`
- `To Date`
- `Appointment Type`

## Was sie tun

- `From Date`
  Das ist der erste moegliche Tag fuer erzeugte Termine.

- `To Date`
  Das ist der letzte moegliche Tag fuer erzeugte Termine.

- `Appointment Type`
  Das waehlt die Art des erzeugten Termins.

## Appointment Type Optionen

- `dentist`
  Beispiele fuer Zahnarzttermine

- `wallbox`
  Beispiele fuer Elektrizitaets- und Wallbox-Kontrollen

- `gas_meter`
  Beispiele fuer Gaszaehler-Kontrollen

- `water_meter`
  Beispiele fuer Wasserzaehler-Ablesungen
