# User Guide Demo Monitoring UI v1.1.0 Patch 6

Version: v1.1.0-patch6
Audience: User
Language: EN

## Google Demo Control now has three main fields

- `From Date`
- `To Date`
- `Appointment Type`

## What they do

- `From Date`
  This sets the first possible day for generated appointments.

- `To Date`
  This sets the last possible day for generated appointments.

- `Appointment Type`
  This chooses the style of the generated appointment.

## Appointment Type options

- `dentist`
  Dentist appointment examples

- `wallbox`
  Electricity and wallbox inspection examples

- `gas_meter`
  Gas meter inspection examples

- `water_meter`
  Water meter reading examples
