# User Guide Demo Monitoring UI v1.2.0

Version: v1.2.0
Audience: User
Language: DE

## Was die Buttons tun

- `Verschieben`
  Laedt die naechsten verfuegbaren Terminzeiten.

- `Bestaetigen`
  Bestaetigt den gewaehlten Slot und startet die finale Buchungspruefung.

- `Absagen`
  Sagt den Termin ab oder stoppt den aktuellen Booking-Pfad.

- `Zurueckrufen`
  Fordert einen Rueckruf oder eine menschliche Uebergabe an.

## Was bei der Slot-Auswahl passiert

Das System erstellt eine temporaere Reservierung.

Dadurch wird der Slot kurzzeitig gehalten, damit ein anderer Nutzer ihn nicht parallel buchen soll.
