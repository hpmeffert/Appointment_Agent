# User Guide Demo Monitoring UI v1.2.0

Version: v1.2.0
Audience: User
Language: EN

## What the buttons do

- `Reschedule`
  Loads the next available appointment times.

- `Confirm`
  Confirms the selected slot and starts the final booking check.

- `Cancel`
  Cancels the appointment or stops the current booking path.

- `Call Me`
  Requests a callback or human handover.

## What happens when a slot is selected

The system creates a temporary reservation.

This means the slot is held for a short time so another user should not book it in parallel.

## What happens if a slot becomes unavailable

The system shows a clear message and offers the next available times.
