# User Guide Demo Monitoring UI v1.2.1-patch1

Version: v1.2.1-patch1
Language: EN

## What is new

There is now a separate `RCS Settings` page.

It helps an operator understand:
- which LEKAB values are already set
- which values are still missing
- if the adapter is ready enough for test messaging

## Important words

- `ready`
  The adapter has enough required values for a test messaging setup.
- `missing fields`
  These are the values still needed before the setup is complete.
- `SMS fallback`
  If RCS is not possible, the system can fall back to SMS.
